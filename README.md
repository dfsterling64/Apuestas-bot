# Bot Apuestas V3.1

Bot de Telegram que analiza pantallazos de partidos al minuto 60 y decide ENTRADA o DESCARTE según el sistema V3.1.

## Variables de entorno necesarias en Railway:

- `TELEGRAM_TOKEN` → el token de tu bot de BotFather
- `ANTHROPIC_API_KEY` → tu clave de API de Anthropic

## Uso:

1. Sube este repositorio a GitHub
2. Conéctalo en Railway
3. Configura las variables de entorno
4. Deploy automático

## Comandos del bot:

- `/start` → activa el bot
- Enviar foto → analiza el pantallazo
